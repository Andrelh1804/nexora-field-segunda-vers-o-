# PROMPT V7.0 — ETAPA 1: IMPLANTAÇÃO ENTERPRISE EM PRODUÇÃO (CLOUD, ALTA DISPONIBILIDADE, ESCALABILIDADE E OPERAÇÃO 24x7)

## CONTEXTO

Você atuará como um **Comitê Executivo de Arquitetura Cloud Enterprise**, composto por:

* Chief Cloud Architect
* Principal DevSecOps Engineer
* Site Reliability Engineer (SRE)
* Kubernetes Administrator (CKA)
* Google Cloud Solutions Architect
* Network Security Engineer
* Database Architect
* FinOps Engineer
* Platform Engineer
* Enterprise Infrastructure Architect

Sua missão é preparar a **NexoraField AI** para operar em **produção corporativa**, garantindo alta disponibilidade, escalabilidade automática, resiliência, segurança, observabilidade e recuperação de desastres.

**IMPORTANTE**

* Não remover nenhuma funcionalidade existente.
* Não alterar regras de negócio.
* Não modificar interfaces públicas.
* Todas as alterações devem ser compatíveis com a arquitetura atual.
* Toda infraestrutura deve ser definida como código (Infrastructure as Code).
* Toda configuração deve utilizar variáveis de ambiente seguras.
* Toda implantação deve ser reproduzível.

Transformar a infraestrutura atual em uma plataforma Cloud Native Enterprise capaz de operar 24 horas por dia, 7 dias por semana, suportando milhares de empresas, técnicos e ordens de serviço simultaneamente.

# CRITÉRIOS DE ACEITE

A etapa somente poderá ser considerada concluída quando:

* Todos os serviços estiverem containerizados.
* A infraestrutura estiver definida como código.
* O ambiente suportar alta disponibilidade.
* O banco possuir backup e failover testados.
* O Redis operar em alta disponibilidade.
* O deploy ocorrer sem indisponibilidade.
* Os testes automatizados forem aprovados.
* O monitoramento estiver ativo.
* Os alertas estiverem configurados.
* Os custos forem documentados.
* Os procedimentos de recuperação forem validados.
* Toda a documentação estiver sincronizada com a infraestrutura implantada.

---

# RESULTADO ESPERADO

Ao concluir esta etapa, a NexoraField AI deverá operar sobre uma infraestrutura **Cloud Native Enterprise**, altamente disponível, resiliente, segura e observável, preparada para crescimento horizontal, implantação contínua, recuperação automática de falhas e operação ininterrupta (24x7). A plataforma deverá estar apta a atender milhares de empresas e técnicos simultaneamente, com governança operacional, monitoramento em tempo real, custos previsíveis e processos de implantação totalmente automatizados. Além disso, o painel **"Realidade do Sistema ⚖️"** deverá refletir automaticamente o estado operacional da infraestrutura com base em métricas reais, servindo como fonte confiável para auditorias, operação e tomada de decisão.
